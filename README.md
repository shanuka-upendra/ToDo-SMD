### Prerequisites

- Java JDK 8 or higher

---

### Installation & Run

1. Clone the repository:
   ```bash
   git clone https://github.com/shanuka-upendra/ToDo-SMD.git
   ```
2. Navigate to the project directory:
```
cd ToDo-SMD
```
3. Build the project:
```
mvn clean install
```
4. Run the Spring Boot application:
```
mvn spring-boot:run
```
5. The application will start on:
```
http://localhost:8080
```
